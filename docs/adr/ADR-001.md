# ADR-001

## Título

Elección de React como framework Frontend

---

## Estado

Aceptado

---

## Contexto

El proyecto JGTravel requiere una interfaz moderna,
multilenguaje,
escalable,
orientada a componentes y preparada para consumir múltiples APIs de terceros.

---

## Decisión

Se utilizará React junto con Vite y TypeScript.

---

## Justificación

- Arquitectura basada en componentes
- Virtual DOM
- Excelente ecosistema
- Fácil integración con Firebase
- Compatible con React Native
- Gran comunidad

---

## Consecuencias

Positivas

- Escalable
- Alto rendimiento
- Reutilización de componentes

Negativas

- Curva inicial de aprendizaje
- Necesidad de comprender Hooks
