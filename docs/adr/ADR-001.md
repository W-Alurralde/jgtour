# ADR-001
## Elección del Stack Tecnológico Base

**Estado:** Aprobado

## Contexto

JGTravel se desarrolla como una plataforma web moderna orientada al turismo para Latinoamérica. El proyecto debe ser escalable, responsive, multilenguaje y preparado para futuras integraciones con servicios externos.

## Decisión

Se adopta el siguiente stack tecnológico:

- React
- Vite
- TypeScript
- React Router
- i18next
- Git + GitHub
- Vercel para despliegue

## Justificación

- Alto rendimiento.
- Arquitectura modular.
- Excelente experiencia de desarrollo.
- Gran comunidad.
- Preparado para crecimiento.

## Consecuencias

La aplicación será desarrollada como SPA utilizando componentes reutilizables y arquitectura modular.
