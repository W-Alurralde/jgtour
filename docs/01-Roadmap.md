# JGTravel - Roadmap

## Visión

Desarrollar una plataforma web de turismo moderna, escalable y multilenguaje para Latinoamérica, integrando servicios de vuelos, hoteles, gastronomía, transporte, alquiler de vehículos y experiencias turísticas.

---

## Sprint 1 – Base del Proyecto ✅

- React + Vite + TypeScript
- React Router
- Internacionalización (i18next)
- Git + GitHub
- Rebranding JGTour → JGTravel

---

## Sprint 2 – Arquitectura 🚧

- Arquitectura del proyecto
- Design System
- Documentación técnica
- ADR
- Refactor de estructura
- Configuración base
